1. 实践具有直接现实性

   1. 实践的三大基本特征：

      - 社会历史性：任何实践活动都处于一定的社会关系之中，具有社会性，社会性决定了其历史性。
   2. 人民群众创造历史的活动的首要的、决定性的影响：经济条件 （`干啥也得有钱啊`）

   4. 唯物主义：物质第一性、意识第二性

2. 人所特有的：**情感、信念、意志**

3. **简单商品生产发展** 和 **资本主义商品生产** 的区别：**有无雇佣工人**
